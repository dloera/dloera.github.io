CIT 82 - Fall 2018

My Project 0

I chose to focus my project for the CIT 82 class Project 0 on the class itself. The class was very helpful in learning a lot of the basics around web technologies. Rio was a very supportive teacher, and the videos were quite helpful. While many of the topics in the class I am already aware of, as I grew up around computers, I still learned quite a bit. This was especially true for learning the gist of HTML/CSS and Bootstrap. I haven't built a website since around 2004 or so and things have changed quite dramatically. Learning CSS was so different than having to do everything with the HTML itself. Also learning how to use github was great. I've been aware of its existence for awhile, but just never got around to how to learn it. Every time I was looking for a project in github, I was clueless as to how I would be able to use it. Now I can figure out how to branch a project if would like to, or just download the files to compile a program. I'm excited to continue along this path and take CIT 85 starting next week, and am very glad that I was able to take this class as a basic before getting into the nitty grittiness of HTML/CSS further. While I'm not sure that I want to be a web developer, I have appreciated all of the help that I've gotten in the class. Cheers!
